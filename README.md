# UNA 🌸

> Plataforma para apoio à saúde íntima feminina no ambiente universitário.

## Sobre o Projeto

O UNA é um projeto desenvolvido na disciplina de Integração e Evolução de Sistemas de Informação com o objetivo de facilitar o acesso de estudantes universitárias a recursos relacionados à saúde íntima feminina.

A proposta surgiu após entrevistas realizadas com estudantes da Universidade Federal de Pernambuco (UFPE), que evidenciaram dificuldades no acesso a absorventes, desconhecimento dos serviços existentes e impactos dessas questões na rotina acadêmica.

O sistema busca conectar usuárias aos pontos de coleta e distribuição de recursos, além de permitir doações, monitoramento de estoque e envio de feedbacks.
